# 📌 Project : ReBuyForMe clone
> rbfy(rebuyforme) 웹 서비스의 UI 구조와 사용자 경험 패턴을 학습하기 위해 제작된 클론코딩 프로젝트입니다.
> 
> swiper 커스터마이징과 gird 속성에 대해 학습하고 활용할 수 있는 기회였습니다.
> 
> 원 사이트와 최대한 비슷하도록 UI를 구현하고자 하였으며, 사용자들의 몰입감 향상에 포커스를 맞춰 제작하였습니다.
> 
> 상업적 목적이 없으며 학습 및 개인 포트폴리오용입니다.

## Development Notes

### _1) Main page_
- 이벤트 배너, 제품 및 아이콘 hover 애니메이션을 통해 사용자가 콘텐츠에 자연스럽게 집중하도록 시각적 몰입도를 강화했습니다.
- Swiper를 활용하여 카테고리별 슬라이드, 자동 전환 등 다양한 패턴의 UI를 구성하여 화면의 다체로움을 높였습니다.
- Tap 메뉴 구현으로 한정된 뷰포트에서 다양한 콘텐츠가 자연스럽게 전환토록 구현하였습니다

### _2) Sub page_ *(메뉴 -> 브랜드 탭 -> 스토리)*
- 스크롤 위치에 따라 Header 상태를 변경하고 ‘go to top’ 버튼을 추가하여 사용자의 시야를 확보하고, 이동 경로를 최소화하여 사용성을 강화했습니다.
- Swiper 커스터마이징으로 브랜드 스토리 이미지를 시간 흐름에 따라 순차적으로 읽히는 인터랙션 구조로 설계하여 브랜드 서사의 몰입감을 높였습니다.
- Gird 속성 활용을 통해 다양한 크기의 이미지를 공간에 맞게 정렬하여 레이아웃 균형을 유지하고 시각적 리듬감을 부여하였습니다.
